System Management Tools

System management tools are determined by scc-smt by means of the contents of the selection file custom/scc-smt-select. All snapshots in a realm are searched for installed software (filesets, depots and packages) that are mentioned in the selection-file. When the configuration file does not exist, no SMT-data is selected and the home page of the realm will not show an SMT-entry.

When you edit the custom configuration file and you install a new version of scc-srv, scc-smt notifies on the generated page that the default configuration file is different and more recent then the specific configuration file in custom. In that case, check the default file for additions/changes from the new scc-srv version or touch the specific file in custom to avoid this message.
